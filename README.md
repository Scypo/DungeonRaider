
 # Game Description
 **Dungeon Rider** is a roguelike game where your goal is to survive as many rooms as possible. 
 Each room is filled with random obstacles and enemies, and every enemy has unique stats depending on their weapons. 
 The game gradually becomes more challenging as you progress through the dungeon.
 
 ---
 ## Controls
- WSAD - walking
- LMB - shooting
- R - reload
- ESC - BuyMenu
- F11 - Fullscreen
## Weapon Types
- AssaultRiffle,
- SMG,
- DMR,
- Sniper
## Upgrades
- health boost
- shield boost
- weapon boost
- medkit

## Build
To build your project, you need to link against [ScypLib](https://github.com/Scypo/ScypLib) and [GLFW](https://www.glfw.org/download). Make sure to set language standard to **C++ 20**
